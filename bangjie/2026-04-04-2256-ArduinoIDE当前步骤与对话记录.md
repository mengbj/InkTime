## 2. 下一步怎么做（按界面一步一步点）

### 第一步：点顶部 `Select Board`

你截图里顶部中间有一个下拉框：

- `Select Board`

现在请直接点它。

### 第二步：搜索并选择板卡

点开后，在搜索框输入：

- `ESP32S3 Dev Module`

然后选择：

- `ESP32S3 Dev Module`

如果弹出的是 ESP32 板包安装提示，也继续允许，因为我们前面本机已经把 Core 装好了，正常应能选到。

### 第三步：确认右上角或顶部已经显示选中的板卡

选中后，顶部不应该再显示 `Select Board`，而应该显示类似：

- `ESP32S3 Dev Module`

---

## 3. 选完板子后，继续设置 Tools

当你完成板卡选择后，再按下面路径逐项设置：

1. `Tools -> Flash Size -> 16MB`
2. `Tools -> PSRAM -> OPI PSRAM`
3. `Tools -> USB CDC On Boot -> Enabled`
4. `Tools -> Upload Speed -> 115200`

如果菜单名字有一点点不一样，以含义一致为准。

---

## 4. 再下一步：选串口

设置完上面四项后，再去：

- `Tools -> Port`

选择：

- `/dev/cu.usbmodem5A470123231`

如果能看到这个串口，就说明 Arduino IDE 已经认到你这块板子。

## 5. 上传前特别提醒

在点击上传前，当前最稳的做法仍然是：

- **先把墨水屏转接板到 ESP32 的所有杜邦线拔掉**
- 只保留 `ESP32-S3 + USB`

因为我们前面已经多次确认，这块板子当前的卡点是：

```text
Failed to write to target RAM (result was 0107: Checksum error)
```

所以这一步是为了优先验证：

- 裸板能不能成功烧录

只要裸板能烧录成功，后面再接回墨水屏就会更稳。

---

## 6. 上传时的动作

点击左上角带箭头的按钮：

- `Upload`

如果卡在 `Connecting...`，就手动按：

1. 按住 `Boot`
2. 短按 `RST`
3. 松开 `RST`
4. 继续按住 `Boot` 1~2 秒
5. 松开 `Boot`
6. 立刻再次点击 `Upload`
