# Arduino IDE 选板与网络报错处理记录

## 1. 当前界面状态

根据本轮最新截图，已经确认：

- Arduino IDE 已打开
- `ink-display-7C-photo.ino` 已打开
- 顶部 `Select Board` 下拉框已展开
- 串口列表里已经出现：`/dev/cu.usbmodem5A470123231`
- 当前右下角提示 `No board selected`

这说明现在已经继续推进到：

- **板子串口已被 Arduino IDE 看见**
- **下一步只差把这个串口绑定到正确板卡 `ESP32S3 Dev Module`**

---

## 2. 右下角报错怎么看

截图右下角报错是：

- `Could not install Arduino AVR platform`
- 下载 `arduinoOTA-1.3.0-darwin_amd64.tar.bz2` 时连接被重置

这不是当前 InkTime 这块 `ESP32-S3` 的核心阻塞。

原因是：

- `Arduino AVR platform` 是给 AVR 板子用的，例如 Uno / Nano
- 我们现在要用的是 `ESP32 Arduino Core`
- 所以这个弹窗 **先不用管**，不影响我们继续选 `ESP32S3 Dev Module`

也就是说：

- **这个错误可以先忽略**
- **不要因此中断当前 ESP32-S3 的设置流程**

---

## 3. 现在应该怎么点

请按下面顺序继续：

### 第一步

点击下拉菜单最下面这一项：

- `Select other board and port...`

### 第二步

进入新窗口后：

1. 在搜索框输入：`ESP32S3 Dev Module`
2. 在板卡结果里选择：`ESP32S3 Dev Module`
3. 在端口里选择：`/dev/cu.usbmodem5A470123231`
4. 点击确认 / OK

如果结果列表很多，优先找名称完全或最接近：

- `ESP32S3 Dev Module`

---

## 4. 选完后的正确结果

选完后，Arduino IDE 顶部应该不再显示：

- `Select Board`

而应显示成类似：

- `ESP32S3 Dev Module on /dev/cu.usbmodem5A470123231`

只要出现这种格式，就说明：

- 板卡选好了
- 串口也绑定好了

---

## 5. 选好板子后的下一步

完成板卡和串口绑定后，再继续设置：

1. `Tools -> Flash Size -> 16MB`
2. `Tools -> PSRAM -> OPI PSRAM`
3. `Tools -> USB CDC On Boot -> Enabled`
4. `Tools -> Upload Speed -> 115200`

然后再准备上传。
