## 下一步应该怎么继续

### 第一步：接回墨水屏
仍按当前项目定义接线：
- `BUSY -> GPIO14`
- `RST -> GPIO13`
- `DC -> GPIO12`
- `CS -> GPIO11`
- `SCLK -> GPIO10`
- `DIN -> GPIO9`
- `3V3 -> 3V3`
- `GND -> GND`

注意：
- 转接板上的 `SCL/SDA` 在这里是 SPI，不是 I2C
- 必须供电 `3V3`，不要接 `5V`

### 第二步：接好屏幕后重新上电
重新上电后继续观察两类现象：
- 串口里是否出现显示初始化日志 / 刷屏日志
- 墨水屏是否出现闪屏、清屏、刷新痕迹，或最终显示内容

### 第三步：如果热点已经出来，可继续配网
既然当前串口里已经看到：
- `softAP result = OK`
- `AP SSID = InkTime-xxxx`

那么后面也可以继续：
- 连接热点 `InkTime-xxxx`
- 打开 `http://192.168.4.1/`
- 填入 WiFi
- 填入服务器地址（局域网 IP，例如 `192.168.1.20:8765`）

注意：
- 不要填 `127.0.0.1:8765`
